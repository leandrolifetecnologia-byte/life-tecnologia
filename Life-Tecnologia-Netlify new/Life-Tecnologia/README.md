# Life Tecnologia

Site institucional premium para Life Tecnologia, pronto para GitHub e Netlify.

## Como subir no GitHub

```bash
git init
git add .
git commit -m "Site Life Tecnologia"
git branch -M main
git remote add origin https://github.com/SEU-USUARIO/life-tecnologia.git
git push -u origin main
```

## Como publicar no Netlify

1. Entre no Netlify.
2. Clique em **Add new site**.
3. Escolha **Import an existing project** para usar GitHub, ou arraste a pasta do projeto.
4. Build command: deixe vazio.
5. Publish directory: `.`

## Personalizações importantes

- Trocar o número do WhatsApp em todos os arquivos: `5500000000000`.
- Inserir fotos reais em `assets/img/` e substituir os placeholders.
- Atualizar e-mail, Instagram, YouTube e domínio.
- Atualizar `sitemap.xml` se usar outro domínio.
